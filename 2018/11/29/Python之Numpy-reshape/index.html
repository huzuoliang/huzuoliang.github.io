Python之Numpy.reshape

2018-11-29
python

numpy.reshape(a, newshape, order=’C’)

  • a : array_like
  • newshape : int or tuple of ints
  • order : {‘C’, ‘F’, ‘A’}, optional

numpy.reshape实际上先对数组ravel，转成一行，然后reshape：

  • np.reshape(a,(2,8), order=’C’)等价于a.ravel(order=’C’)，将数组按行转成[0,1,2,3,4,5,6,…]，然后按行组织（先将行组织好）
  • np.reshape(a,(2,8), order=’F’)等价于a.ravel(order=’F’).reshape(2,8,order=’F’)，将数组按列转成[0,4,8,12,1,5,9,…]，然后按列组织（先将列组织好）
