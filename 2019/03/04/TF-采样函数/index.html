1. 简介

zipfian齐夫分布是一个统计型的经验规律，是美国语言学家Zipf在研究英文单词出现频率时发现的。如果把单词频率从高到低的次序排列，每个单词出现的频率与它的序号乘积近似一个常数。

在语言学中，，现实中有很多现象遵循该定律：

  • 单词的出现频率：不仅适用于语料全体,也适用于单独的一篇文章
  • 网页访问频率
  • 城市人口
  • 收入前3%的人的收入
  • 地震震级
  • 固体破碎时的碎片大小

2. tf.nn.log_uniform_candidate_sampler(zipfian采样)

建议类别分布不均的数据集负采样使用，该函数基于log-uniform(Zipfian)分布采样，每个类别被采样的概率可近似为：，需将类别按出现的频率从大到小排序，class即为类别排序后的index且index越大，被负采样到的概率越小。

图1：zipfian函数
图2：zipfian函数，x,y取对数

函数说明

Args:

  • true_classes: A Tensor of type int64 and shape [batch_size, num_true]. The target classes.
  • num_true: An int. The number of target classes per training example.
  • num_sampled: An int. The number of classes to randomly sample.
  • unique: A bool. Determines whether all sampled classes in a batch are unique.
  • range_max: An int. The number of possible classes.
  • seed: An int. An operation-specific seed. Default is 0.
    name: A name for the operation (optional).

Returns:

  • sampled_candidates: A tensor of type int64 and shape [num_sampled]. The sampled classes.
  • true_expected_count: A tensor of type float. Same shape as true_classes. The expected counts under the sampling distribution of each of true_classes.
  • sampled_expected_count: A tensor of type float. Same shape as sampled_candidates. The expected counts under the sampling distribution of each of sampled_candidates.

函数说明

Args:

  • distortion: 光滑因子The distortion is used to skew the unigram probability distribution. Each weight is first raised to the distortion’s power before adding to the internal unigram distribution. As a result, distortion = 1.0 gives regular unigram sampling (as defined by the vocab file), and distortion = 0.0 gives a uniform distribution.
